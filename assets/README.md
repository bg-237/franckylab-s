## Recommandations

### Optimisation Images
- Utiliser WebP pour les navigateurs modernes
- Fallback JPEG/PNG pour la compatibilité
- Lazy loading activé
- Compression optimale (80-85% qualité)

### Formats Recommandés
- **Logo** : SVG (vectoriel, scalable)
- **Photos** : WebP + JPEG fallback
- **Icônes** : SVG ou Font Awesome
- **Backgrounds** : CSS gradients + particules canvas

### Tailles Optimales
- **Hero images** : 1920x1080px
- **Portfolio** : 600x400px
- **Thumbnails** : 300x200px
- **Logo** : Vectoriel SVG

## Remplacement des Images

Pour personnaliser avec vos propres images :

1. **Remplacer les URLs Unsplash** dans `index.html`
2. **Ajouter vos images** dans le dossier `assets/`
3. **Optimiser** avec des outils comme TinyPNG
4. **Tester** les performances avec Lighthouse
